# 系统配置

![系统配置](../_media/system_setting.png)

> 描述

有 `Mongodb URI`, `Center Address` 和 `Website Address`三个配置项
- `Database`: Mongodb的名字
- `Host`: Mongodb的地址
- `Auth`: 作者的相关信息
- `Options`: mongodb其他配置项

- `Center Address`: cloudtask center的IP地址

- `Website Address`: cloudtask的站点地址

PS：目前数据库只支持mongodb
